## 执行命令

```bash
# 安装依赖
pip install -r requirements.txt

# 配置密钥
cp .env.example .env
# 编辑 .env 填入 API_KEY / SECRET_KEY / PASSPHRASE

# 启动（本地开发）
python run.py

# 访问
http://localhost:8000
```
