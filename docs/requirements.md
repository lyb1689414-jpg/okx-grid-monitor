# 功能需求文档

## 项目概述
欧易（OKX）合约网格自动监控系统。监控多个合约网格策略的运行情况，每日自动统计配对数据并调整止盈金额。

## 核心功能

### 1. 每日配对统计
- **触发时间**: 每日 23:59（Asia/Shanghai）
- **统计内容**:
  - 当日配对次数（每个网格档位买卖都成交算一次配对）
  - 当日配对金额（配对成交总金额）
  - 当日回报率（配对收益 ÷ 投入本金 × 100%）
  - 当日标的涨跌幅（(收盘价 - 开盘价) ÷ 开盘价 × 100%）
  - 当日标的振幅（(最高价 - 最低价) ÷ 开盘价 × 100%）
- **存储**: 计入 daily_statistics 表，一网格一天一条

### 2. 每日止盈修改
- **触发时间**: 每日 00:00
- **计算公式**: 新止盈金额 = 已配对网格收益 + 总投入金额 × 16.14%
- **执行方式**: 自动调用欧易 API 修改网格止盈参数
- **记录**: 每次修改记入 take_profit_history 表

### 3. 数据展示
- 实时仪表盘（汇总卡片 + 网格列表）
- 振幅走势折线图（ECharts）
- 历史统计查询（支持分页和日期筛选）
- 止盈修改记录查询
- 配对明细查询

### 4. 网格管理
- 自动同步欧易活跃网格列表
- 网格状态追踪（运行中/已停止/已关闭）
- 支持多网格同时监控（5个以上）
